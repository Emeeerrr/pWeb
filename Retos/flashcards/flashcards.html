Flash Cards

HTML 5

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para crear páginas web y otros tipos de documentos que se pueden ver a través de un navegador web. Actualmente se encuentra en su versión 5

CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación visual de documentos HTML, XHTML y XML. CSS se utiliza para definir cómo se presentan los elementos HTML en una página web.

Git Hub

GitHub es una plataforma de alojamiento de código fuente y herramienta de control de versiones que permite a los desarrolladores colaborar en proyectos y realizar un seguimiento de las modificaciones en el código.

PHP

PHP es un lenguaje de programación de código abierto utilizado principalmente para crear aplicaciones web dinámicas y sitios web. Se ejecuta en el lado del servidor y es compatible con una amplia variedad de bases de datos.

Etiquetas

Elementos que se utilizan para definir el contenido y la estructura de una página web. Permiten agregar texto, imágenes, enlaces, videos y otros elementos multimedia. Las etiquetas deben estar correctamente anidadas y cerradas.

Android Studio

IDE utilizado para desarrollar aplicaciones para dispositivos Android. Proporciona un editor de código, depurador, emulador de dispositivo y muchas herramientas para la creación de aplicaciones móviles. Android Studio se basa en IntelliJ IDEA y es desarrollado por Google.

API

API significa "Interfaz de Programación de Aplicaciones". Proporciona un conjunto de reglas y protocolos para permitir la comunicación y la integración entre diferentes sistemas y plataformas de software.

Bootstrap

framework de diseño web de código abierto creado por Twitter. Proporciona una biblioteca de componentes HTML, CSS y JavaScript predefinidos para ayudar a los desarrolladores a crear sitios web responsivos y adaptables fácilmente.

Database

Conjunto de datos organizados y estructurados en una computadora o servidor que se pueden acceder, gestionar y actualizar fácilmente. Las bases de datos se utilizan para almacenar y recuperar información de manera eficiente.

DNS

Significa "Sistema de Nombres de Dominio". Es un sistema que se utiliza para traducir los nombres de dominio de las páginas web en direcciones IP numéricas que las computadoras puedan entender y encontrar.

Figma

Figma es una herramienta de diseño de interfaces de usuario basada en la nube que ofrece un editor de vectores, herramientas de diseño de prototipos y colaboración en tiempo real para equipos de diseño.

GIT

Sistema de control de versiones distribuido de código abierto utilizado para rastrear cambios en el código fuente durante el desarrollo de software. Permite a los desarrolladores trabajar en equipo de manera eficiente y mantener un registro de las versiones del código.

HTTPS

Significa "Protocolo Seguro de Transferencia de Hipertexto". Es un protocolo de comunicación seguro utilizado en la web para garantizar la privacidad, integridad y autenticidad de los datos que se transfieren entre un navegador web y un servidor web.

IP

Una dirección IP es un número único asignado a cada dispositivo conectado a Internet que se utiliza para identificar y comunicarse con otros dispositivos en la red. Las direcciones IP son fundamentales para la comunicación en línea y la conectividad de redes.

JavaScript

Es un lenguaje de programación interpretado utilizado principalmente en el desarrollo web para crear interactividad en sitios web y aplicaciones web. JavaScript se ejecuta en el lado del cliente y se utiliza en conjunto con HTML y CSS.

JSON

JSON significa "Notación de Objetos de JavaScript". Es un formato de datos utilizado para intercambiar información entre aplicaciones y sistemas. JSON utiliza una sintaxis legible por humanos.

Neocities

Neocities es un servicio de alojamiento web que permite a los usuarios crear y alojar sus propias páginas web de manera gratuita. Es un servicio popular para la creación de sitios web personales, blogs y portafolios.

SQL

SQL significa "Lenguaje de Consulta Estructurado". Es un lenguaje de programación utilizado en la gestión de bases de datos para la creación, modificación y recuperación de datos. SQL es ampliamente utilizado en el desarrollo web y empresarial.

TLS

TLS significa "Seguridad de la Capa de Transporte" y es un protocolo de seguridad utilizado en la web para cifrar y proteger la comunicación entre un servidor y un cliente, como un navegador web.

Visual Studio

Visual Studio es un entorno de desarrollo integrado (IDE) utilizado para crear aplicaciones de software para Windows, Android, iOS, web y otras plataformas. Proporciona herramientas de desarrollo y depuración y demás
